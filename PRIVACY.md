# Privacy Policy

Markdown Benez is designed to work locally in the user's browser.

## Data Collection

Markdown Benez does not collect, sell, share, or transmit personal data to external servers.

## Data Processed Locally

The extension may process the following data locally when the user triggers an action:

- selected web page content
- selected links
- selected images
- Markdown documents created or edited in the built-in editor
- editor preferences, such as light or dark mode

This data is processed only to provide the extension's copy, edit, preview, and save features.

## Browser Storage

Markdown Benez may use browser storage to save temporary editor drafts and user preferences. This information remains in the user's browser unless the user removes it through browser settings or extension data controls.

## Clipboard Access

When the user chooses a copy action, Markdown Benez writes the generated Markdown content to the clipboard. Clipboard writes happen only after a user-initiated extension action.

## External Services

Markdown Benez does not use external analytics, advertising, tracking, or telemetry services.
